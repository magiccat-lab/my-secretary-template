## 記入例（リモート勤務のエンジニア）

下の値を上のプレースホルダにコピーして、自分用に書き換える。

### Basic info
- **Name**: 山田 健
- **How the secretary calls you**: ケンさん
- **First-person pronoun**: 僕
- **Timezone**: Asia/Tokyo
- **Work**: フィンテック企業のバックエンドエンジニア。Python と Go メイン

### Context
- 都内で一人暮らし。基本は在宅勤務
- 朝 16:00 にUSチームと standup（向こうの朝）
- 4週に1回オンコール当番が回ってくる

### Tone / personality
- ストレート派。「これから〜します」みたいな前置きが嫌い
- 端的な status update が好み。スクショよりリンク派
- 業務外なら雑談OK

### Likes
- メカニカルキーボード、neovim、週末ハイキング
- 個人開発で小さな SaaS をやってる

### Dislikes / pet peeves
- 同じことを2回確認される
- 1行で済む結論を長々と要約される

### Relationship
ラフな後輩〜同僚ポジション。下の名前で呼ぶ。スケジュールは把握してる
